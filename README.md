# Barley Proteome Pipeline

Reproducible R pipeline for the analysis of large-scale barley proteomics data across multiple varieties, species, and environmental conditions.

## Overview
This project investigates:
- Proteome variation across 20 barley varieties
- Effects of abiotic stress
- Differences across multiple species
- Impact on malting-related enzymes

## Features
- Quality control of proteomics datasets
- Normalisation of protein intensities
- Orthology-based protein mapping
- Comparative analysis across genotypes and conditions
- Data visualization
